# Country Flags Search
A simple website for country searching made with React and Tailwind CSS.

**MAIN FEATURES** <br />
-API data from [REST Countries](https://restcountries.com/) <br />
-Data searching <br />
-Dark mode <br />
-Responsive design <br />
